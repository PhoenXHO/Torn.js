# Torn.js

A TypeScript client for the Torn API v2 (in development).

---

## Project Status

This project is in early development. There is no published npm package yet.

---

## Goals
- Type-safe, modern Torn API v2 client for Node.js
- Async/await, error handling, and extensibility
- Full documentation and tests before first release
